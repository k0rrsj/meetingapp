**Версия:** 1.0  
**Дата:** 06.04.2026

---

## 1. Навигационная структура

```
/login                              — Страница входа
/companies                          — Экран 1: Список компаний (главный)
/companies/:companyId/managers      — Экран 2: Руководители компании
/companies/:companyId/managers/:managerId  — Экран 3: Карточка руководителя
/settings                           — Настройки AI
```

## 10. Визуальные индикаторы статусов

### Статус компании / руководителя

| Статус | Цвет | Текст |
|--------|------|-------|
| active / in_progress | зелёный | Активна / В процессе |
| completed | серый | Завершена / Завершён |

### Статус встречи

| Статус | Цвет | Текст |
|--------|------|-------|
| preparation | синий | Подготовка |
| conducted | жёлтый | Проведена |
| processed | оранжевый | Обработана |
| closed | серый | Закрыта |

### Индикатор автосохранения

Отображается в правом верхнем углу карточки встречи:
- `●` серый — нет изменений
- `●` жёлтый, «Сохранение...» — дебаунс активен
- `●` зелёный, «Сохранено» — успешно (3 сек, затем пропадает)
- `●` красный, «Ошибка» — не удалось сохранить

## 12. Адаптивность

Приложение desktop-first. Базовые правила:

| Breakpoint | Поведение |
|-----------|-----------|
| `lg` (≥ 1024px) | Основной целевой размер, двухколоночные блоки |
| `md` (768–1023px) | Однаколоночный layout, карточки на всю ширину |
| `sm` (< 768px) | Базовая мобильная поддержка, все блоки вертикально |
